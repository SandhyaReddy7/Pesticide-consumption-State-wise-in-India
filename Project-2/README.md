Project Overview

This project analyzes the quality of Arabica coffee beans across different countries using data analytics techniques. The goal is to identify key factors that influence coffee quality and help in making data-driven sourcing decisions

---

## Dataset Information

* **Dataset Name:** Arabica Coffee Quality Dataset
* **Total Records:** 207 entries
* **Countries Included:** Ethiopia, Taiwan, Guatemala, Colombia, Brazil, and more


## Key Findings

* **Top Performing Countries:** Ethiopia, Tanzania, Taiwan, Guatemala
* **Lower Performing Countries:** Brazil, Nicaragua
* **High Altitude = Better Quality**
* **Aroma, Flavor, Aftertaste** strongly influence coffee quality

---

## Insights

* African countries consistently produce high-quality coffee
* Taiwan shows strong performance despite smaller production
* Significant variation exists within the same country
* Sensory attributes collectively determine overall quality
